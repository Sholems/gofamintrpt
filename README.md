# Royal Priesthood Tabernacle Website

Official website for Royal Priesthood Tabernacle (GOFAMINT) - A royal community of believers called out to show forth the praises of Him who called us out of darkness into His marvelous light.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript 5.8
- **Styling:** Tailwind CSS 3.4
- **Fonts:** Google Fonts (Inter + Playfair Display)
- **Deployment:** Vercel-ready

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd royalpriest
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
